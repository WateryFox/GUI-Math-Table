I developed this project to explore how graphical interfaces can make simple mathematical tools more accessible and user friendly. Instead of just printing a table to the console, I used Python's Tkinter library to build a multi window application that allows users to choose a specific operation addition, subtraction, multiplication, or division and see a formatted 10x10 result grid. This project helped me understand the event driven nature of GUI programming, where the code waits for a user's interaction before executing specific logic.

The technical core of the application is the management of Tkinter widgets like Labels, Buttons, and Radiobuttons. I used a Grid Layout Manager to organize the math results into a structured table, which required careful control over row and column indices. To make the interface more professional, I implemented a secondary window system using Toplevel. This allows the user to keep the main selection menu open while viewing the generated table in a separate, ivory colored window. I also used color coding red for rows and blue for columns to make the table easier to read at a glance.

One of the key features I implemented is basic data validation. Before generating a table, the program checks if the user has actually selected an operator using a tk.IntVar(). If not, a messagebox warning appears, which is a standard practice in software development to prevent errors and guide the user. I also handled the mathematical logic dynamically; the program uses a single loop structure that switches its calculation method based on the selected operator, demonstrating efficient code reuse. This taught me how to bridge the gap between back end math logic and front end visual representation.

In the future, I plan to add a feature that allows users to customize the size of the table, perhaps generating a 20x20 or 50x50 grid based on their input. I am also interested in adding a "Save to PDF" or "Export to Excel" button so that the generated tables can be used for offline study. Eventually, I would like to experiment with more advanced styling using the ttk (themed Tkinter) module to give the app a more modern, native look on different operating systems.
